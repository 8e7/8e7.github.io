什麼是 Git，Github？

想像你在做一個程式專案，而你可能想要試試看新增一個功能，但是這個功能可能會動到很多個檔案，如果修完出 bug 了，那要變回原本的版本就會非常麻煩… 這種問題造就了 git 的誕生。

Git 是一個版本控制系統，可以幫電腦裡的檔案和資料夾紀錄不同時間的狀態。有了 git，你就能一步步的進行專案的開發，並且即使犯錯也有反悔的方法。

Github 則是 Git 的遠端資料庫，當你把專案上傳到 Github 之後，就能在另外一台電腦上把他下載下來。Github 最強大的是他的協作功能 — 一個團隊可以使用 Github 同時開發大型的專案，並且將不同地方的修改使用 merge 的方式來合併，這就是為什麼 Git 和 Github 在軟體開發界無所不在。
